Serviços

Criação de sites estáticos utilizando HTML e CSS, com foco em organização, semântica, boas práticas e experiência do usuário. Estruturação limpa e código bem documentado.

Desenvolvimento com Python

Desenvolvimento de scripts, automações e lógica de programação utilizando Python. Aplicação de estruturas de dados, algoritmos e fundamentos de programação orientada a objetos.

Desenvolvimento em Java

Criação de aplicações baseadas em Java com foco em lógica, orientação a objetos e organização de código. Experiência com conceitos de POO e estruturação de sistemas.

Desenvolvimento Front-end

Construção de interfaces utilizando JavaScript e TypeScript, aplicando conceitos modernos de desenvolvimento e boas práticas de código.

Banco de Dados

Modelagem e manipulação de bancos de dados relacionais. Criação de tabelas, consultas SQL, normalização e organização eficiente de dados.

Sistemas e Linux

Experiência com ambiente Linux, linha de comando, estrutura de sistemas operacionais e organização de projetos em ambientes Unix-based.
